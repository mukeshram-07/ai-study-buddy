## 🚀 Getting Started

### Manual Setup

```bash
# Clone the repo
git clone https://github.com/Mukeshram-07/ai-study-buddy.git
cd ai-study-buddy

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```
